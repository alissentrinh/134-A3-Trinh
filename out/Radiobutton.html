Radiobutton

Function that creates radio buttons by creating n radio buttons vertically Users must implement the radio buttons into the HTML document, then set how many radio buttons to create, otherwise a default of 2 buttons will be generated, by the consuming code. The text next to each radio button can be set and changed by the consuming code.
